Com usar-ho

Mode Camino:
Click per afegir punts de control.
El camí es genera automàticament amb corbes suaus (Catmull-Rom).
Mode Nivells:
Click per afegir un node.
Click sobre un node existent per eliminar-lo.
Ràtio:
Les coordenades s'exporten normalitzades (0.0–1.0) relatives a la imatge. Flutter les escala automàticament.
Dreceres:
P Camino   N Nivells
Z Desfer   Esc Tancar
Flux:
1. Carrega la imatge de fons
2. Dibuixa el camí
3. Col·loca els nivells
4. Exporta el JSON
5. Envia'l al dev
